How We Started

This club was founded by Shaun Haque and directed by John Van Duinen. H.E.C was founded upon the principle of educating one's peers and giving people a chance to develop a neccessary technical accumen in a school which had no other Tech Oriented classes or clubs.

What We Have Learned

Instruction and assistance administered by Shaun Haque and John Van Duinen.

  • JavaScript

    Members learned about the interaction of web pages with servers and users through JavaScript online tutorials, demonstrations, and projects.

  • Web Development

    Members learned how to develop a website in a short period of time using various online resources.

  • HTML

    Members used HTML to create a calculator and learn how to structure a website.

  • Python

    Members learned how to write basic python code using the Processing library.

  • CSS

    Members learned how to design their own web pages using CSS alongside HTML and JavaScript.

  • App Development

    Members got a crash-course in android app development through instruction and online resources.
